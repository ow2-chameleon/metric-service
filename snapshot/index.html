OW2 Chameleon ~ Metric Service

Units and measurement made simple.

The metric-service is a simple library to manipulate:
  • measures
  • units
  • conversions
  • dimensions

It also defines a set of units, and let you define your own units.

Concepts

As soon as you handle physical data, measures and units enter into the game. Unfortunately, manipulating them is often difficult and not well-understood. OW2 Chameleon metric-service provides a very simple API to manipulates such data.

Measure

A measure is a quantity taken at a specific time, with a potential deviation (delta).

Metric-Service makes creating and manipulating measures easy as pie.

Read more...

Quantities

A quantity is composed by a value and an unit. Quantities are organized by category such as length, mass, illuminance, electric potential. On quantity belonging to the same category are comparable.

Read more

Units

A unit of measurement is a definite magnitude of physical quantity. Lots of units are specified. The library let you reuse defined ones, create new ones and manage conversions.

Read more

Standalone, JavaEE and OSGi-friendly

The library is packaged in an OSGi bundle. Thus it can be used in an OSGi applications, but also in regular Java applications or in JavaEE applications. Just use whatever environment you want.
